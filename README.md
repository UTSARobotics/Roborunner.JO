# Roborunner.JO
Made by Autobots
This is a project Jacob Waren, Prime Rodriguez III, Briana Sepulveda and Abhishek Das.
# What is it?
This Project is a controller controlled bird which moves like an RC car which speaks and acts like a roadrunner in the spirit of Roadrunners attending this university
# What are the parts we used?
### KOOBOOK 1Pcs L298P Shield R3 Motor Driver Module Expansion Board H-Bridge 2A for Arduino UNO 2560
### Low Profile Caster Wheels Set of 4,
### Arduino Uno REV3 [A000066]
### Infrared IR Wireless Remote Control
### TT Motor with Leads DC 3-6V Gearbox Motor
### DFPlayer - A Mini MP3 Player
### SanDisk microSDHC Card 16 gb
### 36 mm Diameter Metal Shell Round Internal Magnetic Speaker
